# Weight Loss Management System

This is a simple Django web application that helps users track their daily weight and monitor their weight loss progress.

The goal of this project is to make weight tracking easy and organized.

## Features

* User signup, login and logout

* Add daily weight (only 1 entry per day allowed)

* Date and time stored automatically

* View weight history with pagination

* Edit and delete weight entries

* Calculate weight difference between two selected dates
